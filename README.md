# WinDECEE — BladeLoop

Interactive online application that visualises the **thermal co-processing of decommissioned wind turbine blades** — a real chemical engineering process for recycling glass-fibre-reinforced polymer (GFRP) blade waste into cement industry feedstock.

Built for **DE-Project Summer Semester 2026** at Otto-von-Guericke University Magdeburg, a cooperative course between FIN (Computer Science) and FVST (Process & Systems Engineering).

🔗 **Live app:** _coming soon_ — will be deployed at `https://<username>.github.io/windecee-bladeloop/`
